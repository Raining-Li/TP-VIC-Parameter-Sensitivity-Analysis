# TP-VIC-Parameter-Sensitivity-Analysis
This project consists of four main parts:（1）Model Performance Metrics: NSE, KGE, and SPAEF;（2）Sensitivity Analysis: Sampling and analysis of parameter sensitivity using methods like Sobol′, Delta test, and Multivariate Adaptive Regression Splines (MARS); (3) RF-SHAP Attribution Analysis: Random Forest-based SHapley Additive Explanations (RF-SHAP); (4) Attribute Similarity(Euclidean Distance) calculation.
